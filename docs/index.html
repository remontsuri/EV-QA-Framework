EV-QA-Framework

ML-powered QA Framework for Electric Vehicle & IoT Battery Testing

MIT License Python 3.10+ 948 Tests 93% Coverage v2.1.3

Features

  • Thermal runaway detection (rule-based + ML)
  • Anomaly detection with Isolation Forest
  • SOH prediction with LSTM
  • Cell imbalance analysis
  • Compliance testing: UN 38.3, IEC 62660, GB 38031
  • Digital twin simulation
  • V2G scenario modeling
  • Prometheus metrics + Grafana dashboard

Quick start:


pip install ev-qa-framework

Or with Docker:


git clone https://github.com/remontsuri/EV-QA-Framework.git
cd EV-QA-Framework
docker compose up -d
